# Button

## Description

This is a `<Button />` component.

Buttons come in two flavors:

<Button color="#eee" bg="#6b41d2">
  Filled
</Button>

<Button color="#eee" border="1px solid #6b41d2" bg="#fff">
  Bordered
</Button>
